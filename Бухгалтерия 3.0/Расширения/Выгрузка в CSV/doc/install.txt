Как установить расширение

1. В режиме предприятия меню "Настройки" - "Параметры" - "Режим технического специалиста"

2. Перейти в функции для технического специалиста

3. Выбрать "Стандартные" - "Управление расширениями конфигурации"

4. Добавить расширение, согласиться с предупреждением проверки безопасности

5. Перезапустите конфигурацию
