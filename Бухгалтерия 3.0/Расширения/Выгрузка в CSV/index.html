Расширение для выгрузки и отправки отчетов в формате CSV.

Описание

Расширение для выгрузки и отправки отчетов в формате CSV. Расширение добавляет формат выгрузки CSV при отправке отчетов через подсистему «Рассылка отчетов». При формировании CSV-файла, симовлы табуляции заменяются точкой с запятой (;).

Документация

Установка расширения

  1. В режиме предприятия меню "Настройки" - "Параметры" установите галочку "Режим технического специалиста".
  2. Перейти в меню функции для технического специалиста.
  3. Выбрать "Стандартные" - "Управление расширениями конфигурации".
  4. Добавить расширение, согласиться с предупреждением проверки безопасности.
  5. Снять галочку "Безопасный режим".
  6. Перезапустите конфигурацию.

Использование

  1. Открыть меню "Администрирование" - "Печатные формы, отчеты и обработки" - "Рассылки отчетов".
  2. Добавить рассылку, указать тип CSV.
  3. Выполнить рассылку или настроить рассписание.
