Как установить отчет

1. Откройте раздел "Администрирование", пункт "Печатные формы, отчеты и обработки"

2. Нажмите на ссылку "Дополнительные отчеты и обработки"

3. Нажмите кнопку "Добавить из файла", кнопку "Продолжить" и выберите файл с отчетом (расширение erf).

4. Укажите размещение отчета, например, раздел "Продажи" и сохраните изменения.
